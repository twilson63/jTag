## jTag jArea

[Return to Documentation](/jTag/docs/index.html)

The jTag jArea function simplifies textarea tag of type text.

### jArea(name, [value], [attributes])

Basic Example:

    jArea("user[comments]");
    
Output:

    <textarea id="user_comments" name="user[comments]"></textarea>

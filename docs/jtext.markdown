## jTag jText

[Return to Documentation](/jTag/docs/index.html)

The jTag jText function simplifies input tag of type text.

### jText(name, [value], [attributes])

Basic Example:

    jText("user[name]");
    
Output:

    <input id="user_name" name="user[name]" type="text"></input>
